# Tea Shop Financial Model & DCF Valuation

## Overview

This project demonstrates the development of a complete financial model for a single-outlet tea shop using Microsoft Excel.

The model includes:

- Historical Financial Statements
- Revenue Schedule
- COGS Schedule
- Payroll Schedule
- Operating Expense Schedule
- Working Capital Schedule
- Capex Schedule
- Depreciation Schedule
- Debt Schedule
- Equity Schedule
- Three-Statement Financial Model
- Discounted Cash Flow (DCF) Valuation
- Ratio Analysis
- Executive Summary Dashboard

---

## Model Features

- Driver-based revenue forecasting
- FCFF valuation
- WACC based discounting
- Terminal Value using Gordon Growth Model
- Linked three financial statements
- Automated balance sheet checks
- Financial ratio analysis

---

## Tools Used

- Microsoft Excel
- Financial Modelling
- Corporate Finance
- DCF Valuation

---

## Key Outputs

- Enterprise Value
- Equity Value
- FCFF
- Working Capital Analysis
- Profitability Analysis
